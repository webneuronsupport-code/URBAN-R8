# Urban R8 - Inmobiliaria y Constructora

Sitio web oficial (Landing Page) para **Urban R8**, una empresa inmobiliaria y constructora ubicada en Zapopan, Jalisco, que hace que tus sueños sucedan.

## 🌟 Características
- **Diseño Premium**: Interfaz moderna, minimalista y con temática de bienes raíces de lujo.
- **Acordeón Interactivo 3D**: Sección de servicios fluida con animaciones nativas.
- **Carrusel Hero**: Hero header impactante con múltiples imágenes.
- **Responsivo**: Optimizado para funcionar perfectamente en computadoras, tablets y móviles.
- **Modo Oscuro Elegante**: Uso de paleta de colores oscuros con acentos dorados (`#facc15`).

## 🛠️ Tecnologías Utilizadas
- HTML5
- CSS3 (Vanilla, Flexbox/Grid)
- JavaScript (Vanilla, ES6)
- Iconos: Phosphor Icons
- Integración con Chatwoot

## 🚀 Instalación y Uso
Al ser un proyecto estático Vanilla, no requiere procesos de compilación complejos.
Simplemente abre el archivo `index.html` en tu navegador web o sirve la carpeta utilizando un servidor local (ej. Live Server).
